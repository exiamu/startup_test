# Production Outbox

This outbox is for repository-specific prompts, reports, and handoff artifacts used to build the Jarvis system itself in this repository.

Use this instead of `.nexus/outbox/` when the artifact is about:
- evolving Jarvis or NEXUS as a product
- planning the next implementation phase for `jarvis-ui`
- researching or summarizing the state of this repository's Jarvis build

Do not use this for adopting-project runtime work. Those artifacts still belong in `.nexus/outbox/`.

## Status

Contents here are mostly historical build artifacts and prompts from earlier Jarvis-building passes.

Keep them for lineage, audit trail, and future reference.
Do not treat them as the active source of truth when a newer direction exists in:
- `production/STATUS.md`
- `production/plans/implementation_plan.md`
- `production/vault/ARCHITECTURE.md`
